Chromosome Read Density

Summary

The Chromosome Read Density module represents a scatter plot between the log of chromosome length and the log of read number. Each blue dot in this log-log plot is a chromosome. If more than one chromosome is depicted, a red line calculated via least square linear regression indicates an estimation for the distribution of these chromosomes and a correlation measure.

Warning

Chromosome Read Density never raises a warning.

Failure

Chromosome Read Density never raises an error.
