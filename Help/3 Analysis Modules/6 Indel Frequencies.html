Indel Frequencies

Summary

The Indel Frequencies module shows the frequencies of deletions and insertions for the first and second reads. If no read is second, only one plot is shown.

Warning

Indel Frequencies raises a warning if the total indel percent is above 0.5%.

Failure

Indel Frequencies raises an error if the total indel percent is above 1%.

Common reasons for warnings

This module raises warnings or errors if the total indel percent is above a certain threshold.
