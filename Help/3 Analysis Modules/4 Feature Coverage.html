Feature Coverage

Summary

The Feature Coverage module attempts to offer an overview of how the features from the selected annotation file or genome are covered within the Sam/Bam file. For each feature, a bar represents the number of times that feature was found in the reads.

Warning

This module never raises a warning.

Failure

This module raises an error if no feature is covered.

Common reasons for warnings

This module can raise an error if no feature is covered within the Sam/Bam file. This module never raises a warning.
