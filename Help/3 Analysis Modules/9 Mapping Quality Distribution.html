Mapping Quality Distribution

Summary

The Mapping Quality Distribution module shows the distribution of the mapping quality for the reads in the Sam/Bam file.

Warning

Mapping Quality Distribution raises a warning if the fraction (percent/read_number) of reads with lowest quality mapping value is below 0.75.

Failure

Mapping Quality Distribution raises an error if the fraction (percent/read_number) of reads with lowest quality mapping value is below 0.50.

Common reasons for warnings

This module can raise warnings or errors when the fraction of reads with lowest quality mapping value is below a certain threshold.
