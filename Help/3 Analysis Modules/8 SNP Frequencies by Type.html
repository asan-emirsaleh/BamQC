SNP Frequencies by Type

Summary

The SNP Frequencies by Type module shows the overall frequencies of SNPs for the first and second reads. If no read is second, only one plot is shown.

Warning

This module raises a warning if the maximum frequency of at least one SNP is above 0.5%.

Failure

This module raises an error if the maximum frequency of at least one SNP is above 2.5%.

Common reasons for warnings

This module raises warnings or errors if the maximum frequency of at least one SNP is above a certain threshold.
