Soft Clip Length Distributions

Summary

The Soft Clip Length Distributions module generates a visualisation for the distribution of soft clips within for the first and second reads.

Warning

This module raises a warning if the number of clips is greater than 25 at any point along the distribution.

Failure

This module raises an error if the number of clips is greater than 100 at any point along the distribution.

Common reasons for warnings

This module raises warnings or errors depending on the number of clips along the distribution.
