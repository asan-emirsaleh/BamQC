SNP Frequencies

Summary

The SNP Frequencies module shows the overall frequencies of SNPs for the first and second reads. If no read is second, only one plot is shown.

Warning

SNP Frequencies raises a warning if the total SNPs percent is above 1%.

Failure

SNP Frequencies raises an error if the total SNPs percent is above 2%.

Common reasons for warnings

This module raises warnings or errors if the total SNPs percent is above a certain threshold.
